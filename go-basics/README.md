# Go Basics

Introductory Go program and module setup, plus a couple of scratch files
exploring the language.

**Stack:** Go, Fiber

- `main.go` — a Fiber HTTP server reading rows from Postgres.
- `concurrency.go` — goroutines and channels: a square-it worker pool and a
  fan-in over two channels with `sync.WaitGroup`.
- `generics.go` — generic `Map`/`Filter`/`Reduce`, a `Number` constraint with
  `Sum`, and `Keys` over a map.

> Archived learning project. Part of [`learning-archive`](../README.md) — kept for reference with its original git history.
