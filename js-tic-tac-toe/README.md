# Vanilla JS Tic-Tac-Toe

Plain-JavaScript tic-tac-toe game.

**Stack:** JavaScript, HTML

Open `index.html` in a browser to play:

- `game.js` — board state, win/draw detection over the eight winning lines,
  click handling, turn switching, and a reset button.
- `index.html` — the 3×3 grid and styles; highlights the winning line.
