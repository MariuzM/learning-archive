# JavaScript Basics

Scratch files of plain-JavaScript experiments — working through the language
feature by feature while learning.

**Stack:** JavaScript

Each file is a standalone scratch run (`node <file>.js`):

- `main.js` — first experiment: sphere-volume calculation.
- `arrays.js` — `map`/`filter`/`reduce`/`find`, plus `chunk`, `unique`, `groupBy`.
- `closures.js` — counter factory, `once`, `memoize`, and `curry`.
- `async.js` — promises, `Promise.all`/`race`, `retry`, and a concurrency-limited `mapLimit`.
- `objects.js` — destructuring, spread, optional chaining, `pick`/`omit`, `Map` counting.

> Archived learning project. Part of [`learning-archive`](../README.md) — kept for reference with its original git history.
